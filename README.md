<h4 align="center">Kods kluso novērotāju AI risinājumam</h4>


## Kas tas ir?

Klusais novērotājs ir kamera ar WiFi savienojumu, kas visuāli redz dūmus un patiesā gadījumā nosūta brīdinājumu ar bildi apsardzei

## Pielietotās tehnoloģijas

- ESP32-cam novērošanai/datu uzņemšanai
- ESP32-S3 pīkstuļa darbināšanai
- N8N workflow datu apstrādei un pārvadei

## Lietošanas instrukcija

Ievietot interneta SSID/paroli un webhook saiti uz kuru ESP32 kameras modulis sūtīs datus (oriģināli domāts prieks N8N saistīšanas).
